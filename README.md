# 钱迹体检器

一个完全在浏览器本地运行的微信 / 支付宝账单分析工具。账单不会上传，打开 `index.html` 就能使用。

在线演示：<https://hxiao6608-ctrl.github.io/money-checkup/>（即使在线打开，账单仍只在当前浏览器中解析）

## 解决什么问题

- 把微信、支付宝导出的 CSV 账单合并成统一流水
- 自动统计支出、收入、结余、消费分类和高频商户
- 标出高额支出和可能重复的小额消费
- 生成一份可直接保存或打印的“月度省钱体检”

## 快速体验

1. 双击打开 `index.html`。
2. 点击“加载演示账单”查看完整结果。
3. 或拖入自己的微信 / 支付宝 CSV 文件；可多选，数据只在当前浏览器页面中处理。

也可以在本目录启动任意静态文件服务器，例如：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 测试

```powershell
node --test tests/app.test.js
```

## 商业化定位

本项目的首单不是卖“记账软件”，而是卖一个明确结果：客户提交账单后，获得隐私友好的消费体检、三条可执行节省建议和一次人工解释。建议首批体验价 19.9 元，标准版 39 元；详见 `SELLING_KIT.md`。

## 隐私

- 默认没有后端、账号、埋点或第三方接口。
- 导入的文件仅在用户设备内解析。
- 刷新或关闭页面后，页面中的账单数据即消失。
- 如提供人工解读服务，应让客户先自行脱敏，并在交付后立即删除文件。

## 开源参考与版权边界

产品方向参考了 Actual Budget（MIT）和 Firefly III（AGPL-3.0）对“本地优先个人财务”的验证，以及 `lyenrowe/bill-parser` 对中国支付平台账单格式的覆盖思路。本仓库没有复制这些项目的代码；核心解析与界面为独立实现，避免把无明确许可证的代码带入商业产品。

## 当前限制

- 支持常见的微信 / 支付宝 CSV 字段；平台未来改变导出格式时需更新字段映射。
- 自动分类依赖关键词，是辅助判断，不是会计或财务建议。
- 当前版本不保存历史数据，也不读取 Excel 文件。
